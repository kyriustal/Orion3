# Orion

Plataforma de automação e gestão com inteligência artificial integrada.

## 🚀 Tecnologias

- **Frontend**: React 19, Vite, Tailwind CSS
- **Backend**: Node.js (Express), TypeScript
- **Banco de Dados**: Supabase
- **IA**: Google Gemini (Principal) & OpenAI (Fallback)
- **Comunicação**: Socket.io

## 🛠️ Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/orion-2.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure as variáveis de ambiente:
   - Copie o arquivo `.env.example` para `.env`
   - Preencha com suas chaves de API e URLs do Supabase

## 💻 Desenvolvimento

Para rodar o servidor de desenvolvimento:

```bash
npm run dev
```

## 🏗️ Build e Produção

Para gerar o build do frontend:

```bash
npm run build
```

Para iniciar o servidor em produção:

```bash
npm start
```
